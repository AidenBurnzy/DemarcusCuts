# Demarcus Cuts

Modern barbershop landing page scaffold featuring a booking calendar, contact form, and social touchpoints.

## Getting Started

Open `index.html` in your browser or deploy via Netlify for a production-ready host.

## Integrations

- Netlify: static hosting & form handling
- Neon: optional Postgres backend for bookings
- n8n: automate notifications or CRM syncs

## File Overview

- `index.html`: Site structure and content
- `styles.css`: Black-and-white visual system with responsive breakpoints
- `script.js`: Interactive calendar logic and form scaffolding
